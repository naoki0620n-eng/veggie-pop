# やさいくだものPOP

野菜・果物などの商品名を入力すると、AI（Claude）が **①特徴 ②食べ方 ③POP用の一言** を生成する、店頭POP作成補助のための静的PWAです。サーバーは不要で、iPhoneのブラウザから直接 Claude API を呼び出します。個人利用を前提としています。

## 特長

- サーバー不要（HTML/CSS/JSのみの静的アプリ）
- スマホ幅に最適化、ホーム画面に追加してアプリのように使える（PWA）
- APIキーは端末のブラウザ内（localStorage）にのみ保存

## 起動方法

ブラウザの制約（Service Worker / clipboard 等）のため、`file://` で開くのではなく、簡易HTTPサーバー経由で開くことを推奨します。

このフォルダ（`veggie-pop/`）内で、いずれかを実行してください。

```bash
# Python がある場合
python3 -m http.server 8000
```

```bash
# Node がある場合
npx serve .
```

その後、ブラウザで `http://localhost:8000` を開きます。

iPhone実機から使う場合は、同じWi-Fiに繋いだPCで上記サーバーを起動し、iPhoneのSafariから `http://<PCのIPアドレス>:8000` を開いてください。恒久的に使うなら、GitHub Pages などの静的ホスティング（HTTPS）に置くと便利です。

## APIキーの取得

1. <https://console.anthropic.com/> にアクセスしてログイン（またはサインアップ）
2. 「API Keys」からキーを発行（`sk-ant-...` の形式）
3. アプリ右上の歯車（⚙️）ボタン → 「Anthropic APIキー」に貼り付けて「保存」

キーは端末のブラウザ内（localStorage、キー名 `anthropic_api_key`）に保存され、外部サーバーには送られません（APIリクエスト先の api.anthropic.com を除く）。

## iPhoneのホーム画面に追加する手順

1. Safariで本アプリのURLを開く
2. 画面下部の「共有」ボタン（□に↑のアイコン）をタップ
3. 「ホーム画面に追加」を選択
4. 名前（POP）を確認して「追加」

ホーム画面のアイコンから、フルスクリーンのアプリとして起動できます。

## 使い方

1. 商品名（例: トマト、シャインマスカット）を入力
2. 「POPを作る」ボタン（またはEnter）を押す
3. 「特徴 / 食べ方 / POPの一言」の3カードが表示される
4. 各カードの「コピー」ボタンでテキストをクリップボードへコピー

## セキュリティ上の注意

- このアプリはブラウザから **直接** Anthropic API を呼び出します（`anthropic-dangerous-direct-browser-access` を使用）。APIキーがブラウザ内に保存されるため、**共有端末や信頼できない端末では使用しないでください。**
- APIキーの利用には料金が発生します。想定外の課金を防ぐため、Anthropicコンソールで利用上限（使用量アラート／上限）を設定することを推奨します。
- キーが漏れた・不要になった場合は、Anthropicコンソールで速やかに失効（revoke）してください。アプリ側は設定画面で空欄にして保存すれば削除できます。
- 本アプリは個人利用を前提としています。第三者に公開・配布する用途には向きません。

## ファイル構成

| ファイル | 役割 |
| --- | --- |
| `index.html` | 画面構造（ヘッダー・入力・結果カード・設定モーダル） |
| `style.css` | スタイル（スマホ幅・緑系配色・大きめタップ領域） |
| `app.js` | ロジック（API呼び出し・JSON解析・表示・コピー・設定・SW登録） |
| `manifest.json` | PWAマニフェスト |
| `sw.js` | Service Worker（アプリシェルのキャッシュ、APIはキャッシュしない） |
| `icons/` | アプリアイコン（192 / 512 PNG） |
